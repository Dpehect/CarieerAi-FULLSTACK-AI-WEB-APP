🎯 KariyerAI

Profesyonel AI kariyer koçu — CV & iş ilanı analizi, gap, roadmap, sohbet. LLM olarak Ollama kullanır; bulut sunucuda (Vercel) çalışmaz.

⚠️ Bu sayfa bilerek statik

Vercel yalnızca bilgilendirme sitesi sunar. Asıl uygulama bilgisayarınızda streamlit run streamlit_app.py ile açılır (Ollama + ChromaDB gerekir).

1) Gereksinimler

  • Python 3.10–3.12 (tercihen)
  • Ollama + modeller: llama3.1:8b, nomic-embed-text
  • GitHub reposunu klonlayın

2) Kurulum & çalıştırma (Windows)

git clone https://github.com/Dpehect/CarieerAi-FULLSTACK-AI-WEB-APP.git
cd CarieerAi-FULLSTACK-AI-WEB-APP
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
ollama pull llama3.1:8b
ollama pull nomic-embed-text
streamlit run streamlit_app.py

Tarayıcı: http://localhost:8501

3) Neden Vercel değil?

  • Ollama sizin PC’nizde çalışır (localhost:11434)
  • Streamlit uzun ömürlü süreç ister; Vercel serverless function ister
  • ChromaDB yerel disk kullanır
